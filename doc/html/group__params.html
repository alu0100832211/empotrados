◆ M6811_DEF_BAUD [1/2]

#define M6811_DEF_BAUD   0x33

SIO default baud rate.

Defines the default baud rate of the SIO. This baud rate is used to configure the M6811_BAUD register.

  • 0x33 -> 1200 baud (8Mhz cpu)
  • 0x30 -> 9600 baud (8Mhz cpu)
