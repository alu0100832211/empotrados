◆ init_temporizador()

void init_temporizador ( int  factorT)

Inicializa los flags del contador. Cada vez que se llama se resetea el contador.

Parameters
factorTFactor de escalado del procesador. Va de 0 a 7. La frecuencia base del reloj (8MHz) se divide por la potencia de dos elevado al factor de escalado. Ejemplo factoT = 7 8MHz/2⁷ = 16μsg
Returns
Parameters
factorTFactor de escalado del procesador. Va de 0 a 7. La frecuencia base del reloj (8MHz) se divide por la potencia de dos elevado al factor de escalado. Ejemplo factoT = 7 8MHz/2⁷ = 16μsg IMPORTANTE: De momento, usar un factor distinto de 7 está dando problemas para calcular el tiempo
