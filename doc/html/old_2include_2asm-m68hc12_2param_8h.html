◆ M6811_USE_COP

#define M6811_USE_COP   0

Use the 68HC11 COP.

Define this if you are using the COP timer. This activate the COP reset while polling and writing on the serial line.

◆ M6812_CPU_CLOCK

#define M6812_CPU_CLOCK   (32000000L)

CPU Clock frequency.

Define the frequency of the oscillator plugged on the processor. The value is in hertz. By default, use 32Mhz.
